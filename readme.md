## Описание
### Настройка
**Отправка в телеграм**<br>Для задания параметров для новых чатов, нужно в `Config.php`:
1. Обновить по необходимости `CHAT_TOKEN_MAP`, указав соответствующий **токен** для **чата**, записав новый `TOKEN_CITY`, по необходимости
2. Чтобы изменить автоопределение чата в тг (`$chatId`) и токена (`$botToken`), модифицируйте `Config::getTgChatConfig()`, добавив новый домен и задав параметры `default`, `utm_source`
### Фичи
**Телеграм**
1. Делать отбор конкретных вопросов из квиза (тильда) больше не надо. Отбор находит вопросы и ответы квиза самостоятельно -- `(new TelegramSender())->checkQuiz()`
2. Если определить чат не получилось, заявка отправится в один из "стандартных" чатов, которые определяются по `$_GET["city"]` и `$_POST["utm_source"]` -- `Config::getFallbackChatId()`

**Логи**<br>
1. Данные теперь собираются в папке `./logs`, где:
   * `error.log` - Ошибки;
   * `contactData.log` - Контактные данные с форм + `$_POST['utm_source']`;
   * `incomingRequests.log` - Все входящие запросы
2. Лог файл, при достижении размера в 5МБ удаляется и создаётся новый*
   * 5МБ - примерно 2-2.5 кк символов / примерно 2000-2500 строк / примерно 1 мес. истории заявок (при ~50 в день)



\* - Фича требует утверждения
